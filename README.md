  # OpenAxes Claude Marketplace

  Official directory of OpenAxes plugins for Claude Desktop and Claude Code.

  ## Install

  /plugin marketplace add openaxes/openaxes-claude-marketplace
  /plugin install openaxes-ia@openaxes-plugins

  Browse `.claude-plugin/marketplace.json` for the full plugin list.

  ## Updates

  /plugin marketplace update

  ## Maintenance

  Each plugin is owned by its source project and pushed via that project's
  CI. Do not hand-edit files under `plugins/` — they are mirrored from the
  source repositories and will be overwritten on the next publish.
